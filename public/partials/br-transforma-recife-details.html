Transforma Recife Rede de Voluntariado

Sobre

Sozinho você pode muito. Unidos podemos muito mais. Este é o lema do Transforma Recife, uma plataforma colaborativa que permite conexões entre ONGs e pessoas que querem participar de algum tipo de trabalho voluntário. Até o primeiro semestre de 2015, 13 mil voluntários foram regitrados em 280 ONGs.

Veja o projeto em www.transformarecife.com.br

Tecnologias

O stack do projeto usa ruby on rails e banco de dados postgresql, que provê uma api para o front-end, que conta com nodejs(express), Less, AngularJS e grunt para geração dos builds.

A modularização do código bem como a injeção de dependências (incluindo injeção dos arquivos) foi feita com angular, o que permitiu ao time trabalhar confortavelmente com várias branches e minimizar conflitos de código durante os mergese fácil integração. O gerenciamento do CSS usa Less, com um sistema de grids usando flex-box.

Resultados

Em relação a performance e manutenibilidade, os resultados técnicos foram surpreendentes. A separação de conceitos (API, front-end) foi crucial, ajudando desenvolvedores em futuras implementações e diminuindo o risco de builds com erros.

Para os usuários, o novo design e funcionalidades tornaram a plataforma mais atrativa e fácil de usar, inclusie em dispositivos móveis, uma vez que o design responsivo não existia na versão anterior.
