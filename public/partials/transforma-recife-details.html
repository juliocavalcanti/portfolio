Transforma Recife Collaborative network

About

Alone you can do a lot. Together we can do a whole lot more. this is the Transforma Recife's motto. Transforma Recife is a collaborative platform that allows the connection between NGOs and people who want to do some kind of volunteer work. Until the first semester of 2015, 13 thounsand of volunteers were registered and around 280 NGOs.

You can check out the project at www.transformarecife.com.br

Technologies

The project stack has on ruby on rails using back-end with postgresql database, which provides an API for the front-end, that uses grunt to generate the builds.

The whole code modularization and dependency injection (of physical files also) was made with Angular, which allowed the team to deal with several branches with minimal code conflicts during the development. The CSS modularization and usage of CSS3 effects was managed by Less and the front-end environment was running over a node.js server.

Results

Regarding to performance and mantainability, The technical outcomes are stunning. The separation of concerns (API,front-end, ) were crucial to help developers on future implementations, decreasing the risks failing builds and other errors.

For the users, the new layout and functionalities became more attractive and easy to use, including the responsive features, which didn't existed at former version.
